# fullstack
fullstack developer dengan database MySQL, Backend dengan Express.js dan Frontend dengan React.js dan Next.js<br>

git clone https://github.com/limfiq/fullstack.git<br>

Backend
<ul>
<li>npm install</li>
<li>cd backend</li>
<li>node index / nodemon index</li>
</ul>
<br>
Frontend
<ul>
<li>npm install</li>
<li>cd backend</li>
<li>node index / nodemon index</li>
</ul>
